# 🚀 빠른 시작 가이드

## ⚡ 5분 안에 시작하기

### 1️⃣ 파일 확인
다운로드한 파일들을 확인하세요:
```
✅ index_upgraded.html
✅ style_upgraded.css
✅ script_upgraded.js
✅ server_upgraded.py
✅ UPGRADE_GUIDE.md
✅ COMPARISON.md
```

---

### 2️⃣ 환경 설정 (최초 1회만)

#### Windows
```cmd
# 1. Python 설치 확인
python --version

# 2. 필요한 패키지 설치
pip install flask flask-cors requests

# 3. 환경 변수 설정 (Gemini API 사용 시)
set MY_API_KEY_GOOGLE=your_api_key_here
```

#### Mac/Linux
```bash
# 1. Python 설치 확인
python3 --version

# 2. 필요한 패키지 설치
pip3 install flask flask-cors requests

# 3. 환경 변수 설정 (Gemini API 사용 시)
export MY_API_KEY_GOOGLE=your_api_key_here
```

---

### 3️⃣ 서버 실행

```bash
# 다운로드한 폴더로 이동
cd 다운로드경로

# 서버 실행
python server_upgraded.py

# 서버 실행 확인
# 콘솔에 다음과 같은 메시지가 표시됩니다:
# * Running on http://0.0.0.0:5000
```

---

### 4️⃣ 브라우저에서 열기

#### 본인 PC
```
http://127.0.0.1:5000
또는
http://localhost:5000
```

#### 동료 PC (네트워크 공유)
```
# 1. 본인 PC의 IP 확인
# Windows: cmd에서 'ipconfig'
# Mac/Linux: 터미널에서 'ifconfig'

# 2. 동료 PC 브라우저에서
http://본인IP주소:5000
# 예: http://10.123.45.67:5000
```

## 📱 기본 사용법

### 1. 프로젝트 정보 입력
```
📁 투자명: 양극재 공장 신축
🏢 업체명: 대금산업
💰 공사금액: 85.5 (슬라이더 또는 직접 입력)
⏱️ 공사기간: 18 (슬라이더 또는 직접 입력)
```

### 2. 체크리스트 생성
```
[📊 체크리스트 업데이트] 버튼 클릭
→ 대시보드 + 모든 테이블 생성
```

### 3. 대시보드 확인
```
📊 총 서류 수: 52개
✅ 필수 서류: 28개
⚠️ 확인 필요: 18개
➖ 미해당: 6개
[도넛 차트로 시각화]
```

### 4. 검색 사용
```
🔍 검색창에 입력:
- "크레인" → 크레인 관련 서류만
- "MSDS" → MSDS 관련 서류만
- "안전관리자" → 안전관리자 관련 서류만
```

### 5. 필터 사용
```
[전체] - 모든 서류 표시
[필수만] - 필수 서류만 표시
[확인필요] - 조건부 서류만 표시
[미해당] - 면제 서류만 표시
```

### 6. 저장하기
```
[💾 현재 상태 저장] 클릭
→ JSON 파일 다운로드
→ 파일명: 체크리스트_프로젝트명_날짜.json
```

### 7. 불러오기
```
[📂 불러오기] 클릭
→ 저장했던 JSON 파일 선택
→ 모든 정보 자동 복원
```

### 8. AI 가이드 (선택)
```
# 상세 문의사항 입력
"크레인 작업이 포함됩니다. 
고소 작업 안전 수칙을 알려주세요."

[✨ AI 공사 가이드] 클릭
→ AI 분석 결과 표시
```

### 9. 인쇄/PDF
```
[🖨️ 인쇄&PDF저장] 클릭
→ 브라우저 인쇄 창
→ PDF로 저장 또는 프린터 출력
```

## 🐛 문제 해결

### 문제: 서버가 시작되지 않아요
```bash
# 해결 1: 패키지 재설치
pip install --upgrade flask flask-cors requests

# 해결 2: 포트 충돌 확인
# 다른 프로그램이 5000번 포트를 사용 중일 수 있음
```

### 문제: AI 가이드가 작동하지 않아요
```bash
# 해결 1: 환경 변수 확인
# Windows
echo %MY_API_KEY_GOOGLE%

# Mac/Linux
echo $MY_API_KEY_GOOGLE

# 해결 2: 서버 로그 확인
# 터미널에서 에러 메시지 확인
```

### 문제: 차트가 보이지 않아요
```
해결: 브라우저 캐시 삭제
1. Ctrl + Shift + Delete
2. 캐시 삭제
3. 페이지 새로고침 (F5)
```

### 문제: 검색이 작동하지 않아요
```
해결: JavaScript 오류 확인
1. F12 (개발자 도구)
2. Console 탭 확인
3. 에러 메시지 확인
```
